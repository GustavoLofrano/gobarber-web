### Requisitos

Ter instalado pelo menos um gerenciador de pacotes do Node, [Npm](https://www.npmjs.com/) ou [Yarn](https://yarnpkg.com/).

### Installing

Executar na raiz do projeto o seguinte comando para instalar as dependências

```sh
yarn install
```

ou

```sh
npm install
```

## 🎈 Uso <a name="usage"></a>

```sh
yarn start
```
